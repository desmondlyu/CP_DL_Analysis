# 🔬 CP Wafer Map & DL Analysis (完全離線單網頁版)

這是一個專為半導體測試數據分析設計的網頁工具。100% 離線執行，無須連網或安裝任何伺服器環境，直接按兩下 `index.html` 即可在瀏覽器中開啟使用。

---

## 🌟 核心功能特色

### 1. 100% 本地離線運作
* **零依賴與極速啟動**：不需要透過 `npm run dev` 啟動伺服器，也完全不用連線到網際網路下載任何外部資源（如 CDN），適合廠區內部嚴格的防火牆與無網路環境。
* **數據安全無虞**：所有運算、資料解析和圖表渲染完全在您的本機瀏覽器中完成，絕不會將任何 wafer 資料傳送到外部網路。

### 2. 晶圓圖（Wafer Map）
* **自動比例校正**：自動適應不同的 grid 尺寸，在網頁上始終維持完美的正圓比例。
* **物理裝飾細節**：精確標示出 300mm 晶圓外圓、底部凹槽 (Notch) 以及 5mm 的 Edge Exclusion 邊緣排除區。
* **參數熱度圖**：選取特定測試參數後，以高對比度的彩虹色帶 (Turbo Colormap) 呈現格點的分佈。

### 3. 雙向框選聯動功能 (關鍵特色)
* **晶圓空間框選（Spatial Filter）**：在晶圓圖上按住滑鼠左鍵並拖曳拉出藍色虛線框，可以篩選出特定區域的 Die。其餘三張圖表（相關性、直方圖、箱形圖）會自動**同步更新**，只分析該框選區域內的資料。點擊右上角的紅色「清除空間篩選」按鈕即可恢復。
* **圖表高亮同步（Highlight Sync）**：在相關性散佈圖、直方圖或箱形圖上拖曳拉框，晶圓圖上對應的點位會立即同步顯示**紅色外框**，方便您快速追蹤異常點在晶圓上的物理分佈。

### 4. 多維度數據統計圖表
* **相關性分析 (Correlation)**：分析兩個測試參數之間的關聯性，自動計算 Pearson R 相關係數，可切換對數尺度 (Log Scale) 並支援顯示線性趨勢線。
* **分佈箱形圖 (Boxplot)**：展示多個選定參數的統計數據（最大值、最小值、中位數及四分位數），利於進行參數間的分佈比對。
* **頻率直方圖 (Histogram)**：展示單一或多個參數的數值頻率分佈，支援 X/Y 軸對數尺度切換。

## 📁 資料格式要求
本工具支援與原桌面端 PyQt 格式一致的文字數據檔：
1. **檔頭忽略**：系統會自動忽略前 5 行的說明與測試資訊。
2. **欄位標頭**：第 6 行必須為欄位名稱，需包含 `<X>` 與 `<Y>` 座標欄位，其餘欄位為測試參數名稱。
3. **資料內容**：第 7 行起為實際數據，數據間以空白或斜線 (`/`) 分隔，若遇到無效值（如 `***`, `N/A`, `NULL`, `NaN` 等），系統會自動識別為空值。

## 🚀 快速使用步驟

### 第一步：開啟程式
直接在專案根目錄中，按兩下開啟 [index.html](file:///C:/D_BACKUP/AI_Project/web_app/CP_DL_Analysis/index.html)。

### 第二步：上傳數據
1. 在左側控制面板的 **「檔案上傳」** 區塊，點擊並選擇您的 Wafer 測試資料檔，或者直接拖曳檔案至該區域。
2. 載入成功後，狀態會顯示「已載入: 您的檔案名稱」，且右側會自動載入預設圖表。

### 第三步：調整控制與篩選
1. **切換控制分頁**：在側邊欄上方點選 **Wafer**、**Corr** 或 **Dist** 來控制不同的圖表參數。
2. **搜尋與選取參數**：在搜尋框輸入關鍵字可快速過濾參數清單，並從下拉選單或多選框選取您想觀察的參數。
3. **參數值域篩選**：在側邊欄下方的 **「參數值篩選」** 區，您可以手動輸入特定參數的 MIN/MAX 來進行值域過濾，或者勾選 `CAT` 分類參數的項目。

## 🛠️ 常見問題排解

* **Q：為什麼上傳檔案後圖表是空的？**
  * 請檢查檔案格式是否正確。特別是第 6 行是否包含 `<X>` 與 `<Y>` 欄位，以及資料行是否與欄位數量對齊。
* **Q：雙擊 `index.html` 會被瀏覽器安全性阻擋嗎？**
  * 本工具為純原生 JavaScript + Canvas 繪製，不使用任何本地 Wasm、ES Modules 或 API Fetch，因此能完全繞過瀏覽器的 CORS 安全限制，在 `file://` 協定下完美執行。
